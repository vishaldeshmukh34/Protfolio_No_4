# ⚡ The Tech Behind the Magic

My portfolio isn't just a website — it's a showcase of **modern frontend engineering and kinetic UI design**, where every animation and interaction has a purpose.

### ⚛️ React JS
Component-based architecture powering a scalable and maintainable application.

### 🎨 Tailwind CSS
A custom design system built with **Midnight Slate and Sky Blue palettes** for modern UI styling.

### ✨ Framer Motion
The **soul of the UI animations**, enabling:

- Spring physics for natural motion
- Staggered animations for smooth content reveal
- Shared layout transitions using `layoutId`

### 📱 Glassmorphism UI
Modern **blurred backgrounds, soft shadows, and subtle borders** to create an elegant interface.
